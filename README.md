# Easy Microarchitecture Level

Easy Microarchitecture Level allows you to check what x86-64 microarchitecture level your cpu supports.
## Supported x86-64 Microarchitecture Levels

- x86-64-v1
- x86-64-v2
- x86-64-v3
- x86-64-v4
## Supported Brands

- Intel
- AMD
